# Instruction

Whenever you are asked to create, update, or output any documentation, artifacts, markdown files, implementation plans, roadmaps, or walkthroughs, you MUST use Simplified Chinese (简体中文).
Do not use English for explanations in these documents, unless it is a specific technical term, variable name, or code snippet that necessitates English.

This applies to:
- `implementation_plan.md` (Implementation Plans)
- `task.md` (Task Checklists)
- `walkthrough.md` (Walkthroughs)
- Any file in the `docs/` directory
- Any `.md` files you create in the workspace.

Remember the user's global rule: "每次回复我都要用中文。" This extends to ALL written documentation and markdown files produced for the user.
